## Getting Started

```
git clone https://github.com/jcding/instagram_bot.git
```

### Prerequisites

Requires:
* Python 3
* geckodriver

```
pip3 install --upgrade pip
pip3 install selenium
pip3 install chatterbot
```

### Installing

All you need to do is:

```
python3 main.py
```
